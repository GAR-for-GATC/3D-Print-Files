#####IC Holding Tower#####
------------------------------------------------------------------------------------------------------------------
This tower holds custom IC curcuit protoboards.  The .ipt files were made in AutoDesk Inventor and were printed using a Makerbot Flash Forage Dual Extrusion machine.
